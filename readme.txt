A Simple Card Flipping Game.

Rules:
1. Every round, a maximum of 3 cards could be fliped by clicking on the cards.
2. If all 3 cards contains the same number, the triplets will disappear and 300 points are rewarded!
3. If not all the cards are the same, the triplets will stay and 100 points are deducted.
4. You have 360 seconds to play, You lose when the time limit is reached.
5. You win if you can clear all the cards!

How to Play:
1. First enter a multiple of 3 in the input space at the bottom of the game window.
2. Click on any of the cards to flip a card, matching 3 of the same card is your objective!
3. Enter a new number to start a new game!
